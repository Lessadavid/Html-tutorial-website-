Data Attributes

Tables and forms are essential elements in HTML used for organizing and collecting data. In this section, I'll detail some advanced techniques for working with tables and forms in HTML.

Table Headers

  • Use the <th> element within the <thead> section to define table headers. This provides semantic meaning and aids in accessibility.
  • Example:
<table>
<thead>
<tr>
<th>Product</th>
<th>Price</th>
<th>Quantity</th>
</tr>
</thead>
<tbody>
<!-- Table data goes here -->
</tbody>
</table>

Table Caption (<caption>):

  • Add a <caption> element to provide a title or description for the table. Captions improve accessibility and clarify table content.
  • Example:
<table>
<caption>Monthly Sales Report </caption>
<!-- Table headers and data -- >
</table>

Table Colspans and Rowspsans:

  • Use the colspan and rowspan attributes to merge table cells horizontally or vertically.
  • Example:
<table>
<tr>
<td rowspan="2">Merged Cell</td>
<td>Cell 1</td>
</tr>
<tr>
<td>Cell 2 </td>
</tr>
</table>

Styling Tables with CSS:

  • Apply CSS styles to tables for custom formatting, such as adjusting cell spacing, borders, and background colors.
  • Example:
<style>
table {
border-collapse: collapse;
width: 100%;
}
th, td {
border: 1px solid #ddd;
padding: 8px;
text-align: left;
}
</style>
