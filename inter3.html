Semantic Elements

Semantic elements in HTML are specific HTML tags that provide meaning and structure to web content. They help define the type of content enclosed within the tags, making it more accessible to both web browsers and assistive technologies, such as screen readers for users with disabilities. Semantic HTML elements are essential for creating well-structured, readable, and search engine-friendly web pages. Below, I'll detail some of the most common semantic elements in HTML5:

<header>

  • The <header> element represents the introductory or containing section of a page or a section. It often contains site branding, navigation menus, and page headings.
  • It has attributes like action and method to define where the form data should be sent and how it should be processed on the server.
  • Example:
<header>
<h1>Website Title</h1>
<nav>
<ul>
<li><a href="/">Home</a></li>
<li><a href="/about">About</a></li>
<!-- ... -->
</ul>
</nav>
</header>

<nav>

  • The <nav> element represents a section of navigation links. It's typically used for site navigation menus, but it can also be used for other types of navigation within a document.
  • Example:
<nav>
<ul>
<li> <a href="/">Home </a> </li>
<li> <a href="/about">About </a> </li>
<!-- ... -- >
</ul>
</nav>

<main>

  • The <main> element represents the primary content of a document. It should be unique within the document and exclude content that is repeated across a site, such as navigation menus.
  • Example:
<main>
<h1>Page Title </h1>
<p>Content goes here. </p>
<!-- ... -- >
</main>

<article>

  • The <article> element represents a self-contained piece of content that can be distributed and reused independently, such as a blog post, news article, or forum post.
  • Example:
<article>
<h2>Article Title</h2>
<p>Article content...</p>
<!-- ... -->
</article>

<section>

  • The <section> element defines a thematic grouping of content. It's used to structure content within a document, creating subsections that are semantically meaningful.
<section>
<h2>Section Title</h2>
<p>Section content...</p>
<!-- ... -->
</section>
