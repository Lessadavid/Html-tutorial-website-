Attributes

HTML, or Hypertext Markup Language, is the foundation of most websites on the internet. It is a language that is used to create the content and structure of a webpage. HTML tags are used to mark up the text and other elements on a webpage, and they are enclosed in angle brackets. HTML allows you to create headings, paragraphs, links, images, and many other types of content on a webpage. One important concept in HTML is attributes. Attributes are used to provide additional information about an HTML element. They are specified within the opening tag of an element and consist of a name and a value. The name of an attribute describes what kind of information it provides, while the value is the actual information that is being provided. For example, the "href" attribute is used to specify the URL of a link, and the "alt" attribute is used to provide a text description of an image. Attributes are an important feature of HTML because they allow you to customize the behavior and appearance of your webpage. By using attributes, you can control how links are opened, specify the size and alignment of images, and even add interactive elements like forms and buttons. Attributes can be used to make your webpage more accessible to users with disabilities, by providing descriptions for images and other non-text content. Overall, attributes are a powerful tool for creating dynamic and engaging webpages.

Attributes Discription
id Provides a unique identifier for an element, allowing it to be targeted for styling or scripting.
class Assigns one or more class names to an element, which can be used for styling or selecting elements with JavaScript or CSS.
style Defines inline CSS styles for an element to apply specific styling rules directly.
src Specifies the source URL for elements like img, script, and iframe to load external content or resources.
href Used in the 'a' (anchor) element to specify the URL to which the link points.
alt Provides alternative text for 'img' elements, displayed when the image cannot be loaded or by screen readers for accessibility.
width and height Determines the dimensions of images and certain other elements.
target Specifies where a linked resource will open, such as _blank for a new browser tab or window.
rel Defines the relationship between the current document and a linked resource, often used in 'link' and 'a' elements.
type Specifies the MIME type of linked resources, like stylesheets and scripts.
value Sets the initial value of form elements like 'input' and 'button'.
name Identifies form elements when submitted to the server, typically used with form fields.
placeholder Provides a temporary hint or example text for form inputs.
disabled Disables an input element, making it unresponsive to user interactions.
checked Initially selects a radio button or checkbox input.
