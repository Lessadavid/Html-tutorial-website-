Formattings

Formatting in HTML refers to the process of defining the appearance and structure of content on a web page. HTML provides a variety of elements and attributes that allow you to format text, images, and other elements to create visually appealing and well-structured web pages. One of the primary ways to format text in HTML is through the use of tags and attributes. Here are some common formatting techniques in HTML:

Formattings Discription
b Defines bold text
em Defines emphasized text
i Defines a part of text in an alternate voice or mood
small Defines smaller text
strong Defines important text
sub Defines subscripted text
sup Defines superscripted text
ins Defines inserted text
del Defines deleted text
mark Defines marked/highlighted text
value Sets the initial value of form elements like 'input' and 'button'.
name Identifies form elements when submitted to the server, typically used with form fields.
placeholder Provides a temporary hint or example text for form inputs.
disabled Disables an input element, making it unresponsive to user interactions.
checked Initially selects a radio button or checkbox input.
